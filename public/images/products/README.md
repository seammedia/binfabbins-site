## Image Sources

These images should be extracted from:
1. The original binfabbins.com.au website assets
2. Wayback Machine snapshots: https://web.archive.org/web/*/https://binfabbins.com.au/
3. Product photos provided by the client

## Image Requirements

- Format: PNG or JPG
- Size: Ideally 800x800px or larger (will be automatically optimized by Next.js)
- Background: Transparent or clean white/light gray background
- Aspect ratio: Square or close to square for best display

## Temporary Placeholders

Until actual images are provided, the site will use placeholder images from public sources or show empty states.
